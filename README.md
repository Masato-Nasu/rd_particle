# Reaction–Diffusion (RD)

- クリック / タップ: その位置に **Seed** を打ってパターンを生成
- `R`: 全リセット（初期化）
- `Space`: 一時停止 / 再開
- `P`: 粒子の可視化を切替（デフォルト非表示）
- 開発者コンソール: `window.kill()` で停止、`window.reset()` で再初期化

> このデモは **Gray–Scott 型の連続場 RD** に、粒子が勾配に沿ってドリフトする“diffusiophoresis 風”の要素を付加したミニマル実装です。**Colorado Boulder の論文の厳密再現ではありません。**
